class Eq a => PSOVect a where

Represents the position and velocity of a particle The four functions are necessary in order to use Points to represent positions and velocities of particles (so we can update a position based on its velocity, update its velocity semi-randomly, etc.)

data PSOCand a

A candidate for a global minimum. Stores both the location of the possible minimum, and the value of the function to minimize at that point.

data Particle a

Particles know their location, velocity, and the best location/value they've visited. Individual particles do not know the global minimum, but the Swarm they belong to does

data Swarm a

A Swarm keeps track of all the particles in the swarm, the function that the swarm seeks to minimize, the parameters, the current iteration (for parameters), and the best location/value found so far

Create a swarm in initial state based on the positions of the particles and the grading function. Initial velocities are all zero.
